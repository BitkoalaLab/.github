# 🔬 BitkoalaLab | 比特考拉实验室 🐨

> **“在比特的世界里，像考拉一样专注地探索。”**
> **"Exploring the world of bits with the focus of a koala."**

欢迎来到 **BitkoalaLab**。这是一个专注于个人技术实验、私有化部署方案以及 Web 工具开发的极客空间。我们致力于通过代码提升数字生活的效率，同时坚守数据主权与用户隐私。

---

### 🛡️ 实验室核心研究方向 (Research Focus)

* **Web 基础设施**: 专注于构建个人技术博客、短链接服务及高性能图床系统。
* **网络穿透与架构**: 深度实践 FRP、Cloudflare Tunnel、Tailscale 及 OpenClash 等技术，实现安全高效的内网穿透与 NAS 访问。
* **数据分析与隐私**: 在尊重用户隐私的前提下，探索 Umami、GTM 及 CookieYes 等分析工具的合规化集成。
* **开源规范**: 实验室所有公共项目坚定拥护 **MIT License**，鼓励自由分享与二次创作。

---

### 🚀 旗舰项目 (Featured Projects)

#### 🖼️ [PicKoala](https://github.com/BitkoalaLab/PicKoala)
**PicKoala** 是实验室的核心项目，旨在打造一个简洁、安全且易于扩展的个人图床服务。它不仅是图片的集散地，更是我们对高效存储与优雅展示的视觉表达。

---

### ⚙️ 实验室技术栈 (Tech Stack)

* **Backend**: MinIO, Node.js, Python.
* **Network**: Tailscale, FRP, Cloudflare.
* **Analytics**: Umami, Google Analytics.
* **Tools**: Docker, Git, Duplicati (NAS Backup).
